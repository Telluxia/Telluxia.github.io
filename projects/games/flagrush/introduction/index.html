FragRush

仲間と共にフラッグを奪い、勝利をつかめ！

Overview

フラッグラッシュは、チーム対抗のフラッグ奪取ゲームです。仲間と協力し、相手のフラッグを奪って自分たちの得点ゾーンに運びましょう！

Rules

Base

チーム編成

各チーム6人で編成。
役職は自由に選択可能（一部制限アリ）。

コート分け

体育館の中央でコートを分割し、各チームの端に得点エリアを設定。

フラッグ配置

各チームのエリアに3本のフラッグを自由に配置。

ゲーム目的

相手チームのフラッグを奪い、自チームの得点ゾーンに運びます。

ラウンド時間

前半3分、ハーフタイム1分、後半3分。

勝敗条件

得点エリアのフラッグ数、または生存者数で勝敗が決定されます。
全フラッグ奪取か、敵チーム全滅で即勝利。

Down

ダウン方法

暴走族や魔術師の攻撃でダウンします。
魔術師のアイテム攻撃がキャッチされると無効。

ダウン状態

ダウンしたプレイヤーはその場に体育座りで待機。
僧侶の助けが必要です。

Kamae

ハーフタイムに、各チームに対してランダムでカマエが決定されます。後半戦は決定されたカマエをして戦います。

カマエ一覧

Roles

運び屋

相手チームのフラッグを運び、自チームの得点ゾーンに運びます。

１～２人

回収屋

運ばれた自チームのフラッグ回収します。

１～２人

暴走族

相手チームのプレイヤーにタッチしてダウンさせます。

暴走族

ボールなどのアイテムで相手を攻撃します。
アイテムをキャッチされると無効になります。

僧侶

ダウンした仲間をタッチで復活させます。
